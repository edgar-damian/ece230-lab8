## Lab Questions

### In plain English describe the function and use of a multiplexer.
The purpose of a multiplexer is to guide the flow of many inputs into 1 outputs. For example could be an HDMI in a TV, there can be many HDMI cables inputed into the TV but it only displays 1 of those. 

### In plain English describe the function and use of a demultiplexer.
A demultiplexer does the opposite, it takes one input and routes it to 1 of many outputs. For example, in a car you can guide where the air flows (top, down, ect).


### What other uses might these circuits have? (Think Shannon’s)
These circuits can have many possibilities, with Shannon's expansion, we can use boolean algerbra to break down equations and re create them using muxs. Muxes can make up AND, OR, and NOT gates if needed.
